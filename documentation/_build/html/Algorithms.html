fit(max_iters, obj_func, grad_func, args, step_function, weights=None, tolerance=None, norm=<function norm>, print_status=True)[source]¶

fit gradient descent to data

Parameters

  • max_iters (integer) – number of epochs

  • obj_func (function) – function reference for the objective function to be used

  • grad_func (function) – gradient of objective function

  • args (python list) – agruments for both obj_func and grad_func

  • step_function (function with one parameter taking an integer representing epoch number) – function reference to calculate next step size or learning rate

  • weights (numpy array) – optional argument to use pre-determined weights. zero array by default

  • tolerance (float) – optional stopping condition

  • norm (function) – optional function reference to the norm used to find iterate error, defaults to euclidean norm

  • print_status (boolean) – optional argument to print out status updates

Returns

returns [weights, terminating iteration, objective function values] if tolerance set. Otherwise

[weights, objective function values] :rtype: python list
